Navigating the site:-

The Games Tab

In the games tab, you will be able to see all the games for which reviews have been written. You can click on any title that you want to read a review on and you will be redirected to the respective page.

The Consoles Tab

The consoles tab is a more specific game review search tab. Here you can select the console for whose game you want to read a review about.
